Customer Orders



Customers can add their orders using online upload tool or using Customer API system. When a new order saved into the database, the system automatically try to process the order.

Order with API



  • POST | {URL}/api/orders
  • GET | {URL}/api/orders
  • GET | {URL}/api/orders/{order}

Customer Portal Orders List





Customer can track each order. Again the customer can see why an order failed. This can be a wrong SKU, not enough stock level or not existing the product in customers price or stock group. So, the customer can add a new order with corrected file.
