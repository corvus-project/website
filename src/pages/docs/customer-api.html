Customer API

You may re-generate the customer APO from customer view page. To access this page, please select Account item from left menu. From grid, search the customer with customer number, name or email.







The customer API key is important for your customers to track their orders and create new orders.





API Routes for Customer



  • POST | {URL}/api/orders
  • GET | {URL}/api/orders
  • GET | {URL}/api/orders/{order}
  • GET|HEAD | {URL}/api/products
  • GET|HEAD | {URL}/api/products/{product}
